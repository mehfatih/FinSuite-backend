# PDF Fonts

These WOFF2 files are bundled and embedded into every PDF generated by `src/services/pdf/pdfRenderer.ts` via local `file://` `@font-face` rules. No Google Fonts CDN at PDF render time — Sprint D-2 hard constraint.

## Sources

Both font families fetched via the `@fontsource` CDN (`cdn.jsdelivr.net/npm/@fontsource/...`) — community-maintained packages of the upstream font releases.

- **Inter** — Rasmus Andersson, https://rsms.me/inter/. Open Font License (SIL OFL).
- **IBM Plex Sans Arabic** — IBM Design, https://github.com/IBM/plex. Open Font License (SIL OFL).

## License

Both families are **SIL Open Font License v1.1**. The license permits unmodified or modified bundling, embedding into documents, and redistribution. Embedding in generated PDFs is explicitly allowed and required for proper rendering on systems without these fonts installed.

The full SIL OFL text is preserved in each font's upstream repository — this README satisfies the "include attribution" requirement of the OFL for redistribution in compiled / embedded form.

## Refreshing

If newer font versions are needed, re-run the original download:

```bash
cd src/services/pdf/assets/fonts
for w in 400 500 600 700; do
  for sub in latin latin-ext; do
    curl -sSL -o "inter-${sub}-${w}.woff2" \
      "https://cdn.jsdelivr.net/npm/@fontsource/inter@5/files/inter-${sub}-${w}-normal.woff2"
  done
  curl -sSL -o "ibm-plex-sans-arabic-${w}.woff2" \
    "https://cdn.jsdelivr.net/npm/@fontsource/ibm-plex-sans-arabic@5/files/ibm-plex-sans-arabic-arabic-${w}-normal.woff2"
done
```

Verify magic bytes (every file should start with `wOF2`):

```bash
for f in *.woff2; do head -c 4 "$f" | xxd | head -1; done
```
